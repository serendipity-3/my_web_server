# 加密 HTTP 的数据，我的数据不给你看——HTTPS 

## 背景
我用了 HTTP 发送我的用户名和密码。  
这个 HTTP 在各个路由器中间传输。  
当我的 HTTP 数据包路过你的路由器时，你抓住了我的 HTTP 数据包，然后偷偷拿到了我的用户名和密码，用我的身份干坏事。  
结果你干坏事岁月静好，我却背锅负重前行。  
为了不让这种情况发生，所以要升级 HTTP。
升级后的 HTTP 能加密数据，叫 HTTPS(Hypertext Transfer Protocol Secure)。

最后让权威机构或自己的私钥用它的密钥算一个数字签名得到签发的证书文件(公钥 + 服务器信息 + 数字签名)。
